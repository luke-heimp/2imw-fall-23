Luke Heimpel Storybook Process

Original Idea

Initially, for this project, I wanted to make a game. In this game, the player would use clues to fill out the username and password to access a special HTML page. I designed the puzzle as a desk the player would explore, using bits and pieces of information to figure out the login credentials. I liked this idea, but unfortunately, it would rely on JavaScript to run the password input. I therefore had to scrap this idea.

Sad Tortoise Story

After some feedback from Prof. Ha, I decided to change my idea. I came up with a very simple interactive story that would take advantage of the webpage medium. I drew a sad tortoise using CSS. The viewer is asked to make this tortoise happy by selecting one of three icons at the bottom of the page: money, knowledge, or love. Each icon takes the viewer to a new HTML page. The money and knowledge pages still show a sad tortoise, but the love page sees the tortoise smiling for the first time, a happy ending. I learned a lot through the process of making drawings with CSS and HTML. This project was very beneficial to my learning and I hope to use these new skills in future projects.
